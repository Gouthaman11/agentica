# Problem Statement

Build a professional financial web application that can:

- Accept uploaded bank statement PDFs
- Extract transaction data into a structured format
- Store extracted data in a database
- Present clean dashboard and analytics views
- Help users understand income, expenses, payees, recurring transactions, and anomalies
- Allow download of structured spreadsheet exports for further use

## Solution Direction

This project is organized into two main layers:

- `frontend/` for the customer-facing React experience
- `backend/` for extraction, data processing, storage, and APIs

## Expected Outcomes

- Faster conversion of statements into usable structured records
- Better visibility into spending behavior
- Cleaner operational workflow for uploads, analysis, export, and file management
