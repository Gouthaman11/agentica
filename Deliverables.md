# Deliverables

## Completed

- React + Tailwind frontend with dashboard, analytics, alerts, transactions, documents, and settings pages
- Express backend with MySQL integration
- AWS S3 upload support for statement PDFs
- AWS Textract-based extraction flow
- Structured transaction persistence in SQL
- Analytics and payee breakdown generation
- Excel spreadsheet export for uploaded statements
- Uploaded file delete support from the documents page

## In Scope Artifacts

- `frontend/` application
- `backend/` API and extraction pipeline
- Shared API contracts in `backend/shared/`
- Deployment configuration via `netlify.toml` and `vercel.json`
